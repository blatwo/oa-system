# 瀚高工作记录系统 — 用户操作手册

| 版本 | V5.0 |
| 日期 | 2026-06-10 |

---

## 1. 快速开始

```bash
cd oa-system
python server.py          # http://localhost:8089
```

---

## 2. 模块操作指南

### 2.1 录入工作 / 2.2 工作记录列表 / 2.3 项目工时统计

（同上版本。）

### 2.4 待办事项

- 项目下拉精确筛选
- 勾选完成/取消
- 项目待办 ≥5 条自动标「积压」

### 2.5 工作积压

自动分析："严重积压"/"有积压"/"正常"，进度条可视化。

### 2.6 加班记录 🆕

DataGrid 分页表格，字段：日期、签到、签退、工时、调休、报销、餐补、交通、工作内容。

- 分页：底部可选 15/25/50/100 行
- 排序：点击列头（默认按日期降序）
- 隔行换色 + 悬停高亮
- 头部统计：`共 X 条 · 累计 Xh · 餐补 X · 交通 X`
- 弹窗新增/编辑，支持删除确认

### 2.7 AI 智能分析

（同上版本：选择模型 → 分析模式 → 开始分析，Markdown 渲染结果。）

### 2.8 项目管理

新增/编辑/删除项目。

### 2.9 销售管理 🔄

手动添加关注的销售人员。Tab 2 自动统计每人负责的项目数。

### 2.10 用户画像

每个项目一份画像：商务面貌 / 技术面貌 / 痛点 / 关键人员 / 期望 / 备注。

### 2.11 产品规划

规划方案与产品目录三级分类关联。

### 2.12 版本发布

版本发布与知识产权（软著/专利/商标）关联。

### 2.13 兼容证明管理 🆕

按项目追踪兼容证明：应用系统、是否已做、当前环节（7 种状态）、开具/适配人、证明/适配数量、认领人。

### 2.14 字典维护（子菜单）

- 常规字典 / 产品目录 / 服务目录

### 2.15 LLM 模型管理

配置 OpenAI 兼容 API 模型（测试连接 + 多模型切换）。

---

## 3. 数据保护

每次启动 `python server.py` 自动备份 DB（保留最近 3 份）。备份文件：`oa_data.bak.YYYYMMDD_HHMMSS`。

---

## 4. 常见问题

| 问题 | 答案 |
|------|------|
| 项目下拉框没有？ | 先去"项目管理"新增 |
| 下拉选项不够？ | 字典维护添加 |
| 页面白屏？ | 确保 http://localhost:8089 |
| 换电脑迁移？ | 复制 oa_data.db |
| AI 分析失败？ | 检查 LLM 端点完整 + 测试连接 |
| 数据被删了？ | 用最近备份替换 oa_data.db |
