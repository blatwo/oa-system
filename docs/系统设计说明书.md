## 1. 系统架构

```
浏览器 (React SPA, Vite 构建)
    │ fetch() JSON over HTTP
    ▼
FastAPI Server (Python, Uvicorn)    ← 静态文件托管 (dist/)
    │ sqlite3 + urllib
    ▼                          ▼
oa_data.db (SQLite)          OpenAI 兼容 API
                                   (DeepSeek 等)
```

启动时 `init_db()` 自动：备份现有 DB → 建表/加列（不删库）→ 种子数据仅新表。

## 2. 数据库设计（12 张表）

### work_records
id TEXT PK, date TEXT, project TEXT, env/stage/product1-3/service1-2/method/status/importance/urgency/difficulty, bsc TEXT, hours REAL, content/remark/todo TEXT, todo_done INTEGER

### overtimer 🆕
id INTEGER PK, date TEXT, sign_in/sign_out TEXT, hours REAL, comp_leave/meal/transport TEXT, content TEXT, reimburse TEXT (Y/N), remark TEXT

### compatibility_certs 🆕
id INTEGER PK, project TEXT, app_system TEXT, completed TEXT (Y/N), stage TEXT (7 状态), issue_date TEXT, issued_by TEXT, cert_count/adapt_count INTEGER, adapted_by/claimed_by TEXT, remark TEXT

### 其他表
dictionaries, projects, sales（无种子数据）, llm_profiles, personas, product_catalog, service_catalog, product_plans, product_releases

## 5. 数据保护机制 🛡️

init_db() 启动流程：
1. **备份**：DB 存在 → 复制为 `.bak.YYYYMMDD_HHMMSS`，保留最近 3 份
2. **建表**：`CREATE TABLE IF NOT EXISTS`（已有表不动）
3. **加列**：`ALTER TABLE ADD COLUMN` try/except（已有列不动）
4. **种子**：`INSERT OR IGNORE` 仅对新表生效
